# Docente

Docente es una aplicación web instalable para apoyo docente. Permite registrar alumnos, asistencia, actividades, exámenes, conducta, participación, reportes y respaldos locales.

## Uso

Esta versión está preparada para publicarse con GitHub Pages.

Archivos principales:

- `index.html`: aplicación principal.
- `manifest.webmanifest`: configuración para instalación como app.
- `sw.js`: funcionamiento básico sin conexión.
- `icon.svg`: icono de la aplicación.

## Instalación En Celular O iPad

Después de publicarla en GitHub Pages:

- Android/Chrome: abrir el enlace, tocar menú y elegir instalar app.
- iPhone/iPad/Safari: tocar compartir y elegir agregar a pantalla de inicio.

## Datos

Los datos se guardan localmente en el navegador del dispositivo. Para mover información entre dispositivos, usar las opciones de exportar e importar respaldo dentro de la app.
